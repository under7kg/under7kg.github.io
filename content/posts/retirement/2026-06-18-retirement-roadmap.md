# 🟢 PHASE 1：種を蒔く（30〜40歳）{#1}

## フェーズ1の目的と戦略方針
🎯 ミッション：「複利の時間を最大限に買う」

📌 30歳に投資した1万円 → 年7%・30年後に約7.6万円
📌 40歳に投資した1万円 → 年7%・20年後に約3.9万円

→ 今すぐ始め、1円でも多く積み立てることが最優先。
→ 資産配分の悩みより、積立額の最大化の方が圧倒的に重要。

### 🪣 フェーズ1のバケツ配分目標

| バケツ      | 配分比率        | 内容               |
| -------- | ----------- | ---------------- |
| 🌱 成長バケツ | **90〜100%** | 全世界株式・米国株式インデックス |
| 🔗 繋ぎバケツ | 0〜5%        | まだ不要。特別支出用の積立程度  |
| 💴 現金バケツ | 別枠管理        | 投資とは別に生活費6ヶ月分を確保 |

## 🌱 30歳：スタートダッシュの年

### ⚡ 最初の90日でやること

**📅 Month 1：土台を作る**

```
Week 1-2：口座開設
□ SBI証券 or 楽天証券で新NISA口座を開設
□ iDeCo口座も同時に開設（節税効果大）
□ 家計の現状把握（月収・月支出・現在の貯蓄額を記録）

Week 3-4：💴緊急資金の確保
□ 生活費6ヶ月分（25万円 × 6 = 150万円）を
  高金利普通預金 or 個人向け国債に分離保管
□ これが「絶対に投資に回さないお金」
□ 緊急資金が未達なら、まずここを優先
```

**📅 Month 2：自動積立の設定**

```
□ 新NISAつみたて投資枠：月5〜10万円を自動設定
  └ 🌱成長バケツ商品：eMAXIS Slim 全世界株式（オール・カントリー）
□ iDeCo：月2.3万円（会社員の上限 ※企業年金なしの場合）
  └ 同様に全世界株式インデックス
□ 給与口座→証券口座の自動振替を設定
  💡 自動化で「忘れること」が最強の戦略
```

**📅 Month 3：支出構造の最適化**

```
□ 固定費の見直し（通信費・保険・サブスク）
  🎯 目標：月3〜5万円の捻出
□ 捻出した分を積立額に上乗せ
□ ライフプラン表を作成（結婚・子育て・住宅など想定費用を可視化）
```

### ✅ 30歳時点のチェックポイント

```
✅ 💴緊急資金150万円（6ヶ月分）が別口座に確保されている
✅ 新NISAの自動積立が毎月実行されている
✅ iDeCoが設定されている
✅ 家計の月次収支がプラスになっている
✅ 目標資産進捗管理のスプレッドシートを作成した
```

## 📈 31〜35歳：積立強化の5年間

### 📅 毎年1月：新年度リセット作業（所要時間：2時間）

```
【Step 1：📊 資産棚卸し】
□ 🌱成長バケツ（新NISA）の時価総額を記録
□ iDeCo口座の時価総額を記録
□ 特定口座・預貯金を含めた総資産額を計算
□ 目標資産ペース表と比較

【Step 2：📈 インフレ調整】
□ 前年のCPI（総務省HP）を確認
□ 60歳時の目標生活費を上方修正
  例）20万円 → 20万円 × 1.02 = 20.4万円

【Step 3：💰 積立額の見直し】
□ 昨年の昇給・ボーナスを反映し積立額を増額
□ 新NISAつみたて枠：月10万円（年120万円）を目指す
```

### 📅 毎年7月：中間確認（所要時間：30分）

```
□ 資産残高が年初計画の±15%以内か確認
□ 大きくズレている場合は原因を分析
  （📉 相場要因なら放置、💸 積立不足なら増額）
□ ライフイベント（転職・結婚など）の影響を再確認
```

### 🎯 年齢別・目標資産マイルストーン

```
【📊 年7%運用・月10万円積立の場合】

🌱 30歳時点：      0円（スタート）
   31歳（1年後）： 約  125万円
   32歳（2年後）： 約  261万円
   33歳（3年後）： 約  408万円
   34歳（4年後）： 約  569万円
   35歳（5年後）： 約  745万円 ← 🚩第1チェックポイント

月 5万円なら → 35歳時 約372万円
月15万円なら → 35歳時 約1,117万円
```

### 🚩 35歳：第1チェックポイント（フェーズ1前半の総括）

```
【📊 資産面】
□ 総資産が目標マイルストーンの80%以上あるか
□ 新NISA累積投資額が300〜600万円に達しているか
□ 💴緊急資金は生活費6ヶ月分を維持できているか

【🏠 生活面】
□ 住宅購入計画（賃貸継続 or 購入）の方針が決まっているか
□ 子育て費用の見込みをライフプランに反映したか
□ 保険の見直し（掛け捨て生命保険・医療保険）は済んでいるか

【💼 キャリア面】
□ 収入増の見通しはあるか（転職・昇進・副業）
□ 月積立額を10万円以上に引き上げる道筋があるか
```

## 💼 36〜40歳：収入ピークを積立に直結させる

### 🔑 最重要戦略：収入増加分は生活費に使わない

```
【💡 黄金ルール】
昇給・ボーナスアップ分の50%以上を積立に回す

例）
35歳 手取り月収：35万円 → 積立10万円（28.6%）
40歳 手取り月収：45万円 → 積立15万円（33.3%）
                +10万円増収のうち5万円を積立増額

これを「ライフスタイル・インフレーション」の抑制という
```

### 🚩 40歳：第2チェックポイント（フェーズ1最終確認）

```
【📊 目標資産額の確認】

月10万円・年7%・10年積立の場合：
40歳時点の目標資産 ≈ 約1,740万円（月15万円なら ≈ 約2,610万円）

【達成度別・対応方針】

🟢 達成率100%以上 → フェーズ2に進む（余裕あり）
🟡 達成率80〜99%  → 積立額の増額を優先しフェーズ2へ
🟠 達成率60〜79%  → リタイヤ年齢を1〜2年後ろ倒し検討
🔴 達成率60%未満  → ファイナンシャルプランナーに相談推奨
```

```
【✅ フェーズ1終了時の状態チェック】

必須条件：
✅ 総資産1,500万円以上（月10万積立想定）
✅ 新NISA累積投資額800万円以上
✅ 💴緊急資金が生活費6〜12ヶ月分ある
✅ 住宅・教育費などの大型支出計画が確定している

理想条件：
⭐ 総資産2,000万円以上
⭐ 月積立額が15万円以上に増額できている
⭐ 副収入（配当・副業）が月1万円以上ある
```

# 🟠 PHASE 3：収穫の準備（50〜60歳）{#3}

## フェーズ3の目的と戦略方針

```
【🎯 このフェーズのミッション】

「ゴールへの着陸準備」

残り10年で資産を目標額まで積み上げながら、
リスクを段階的に下げる「グライドパス」を設計する。

→ 積立を継続しながら、徐々に🌱成長バケツ比率を下げる
→ 💴現金・🔗繋ぎバケツを本格的に積み上げる
→ リタイヤ後の生活設計を具体化する
→ 年金・税金・社会保険の戦略を固める
```

### 📉 グライドパス：成長バケツ比率の段階的逓減

```

💡 毎年1〜2%ずつ🌱成長バケツ比率を下げるイメージ
```

# 🔴 PHASE 4：収穫期（60歳〜）{#4}

## フェーズ4の目的と戦略方針

```
【🎯 このフェーズのミッション】

「資産を枯らさずに豊かに生きる」

→ 💴現金バケツから淡々と引き出す
→ 年1回だけリバランス・補充を行う
→ 🌱成長バケツは市場が暴落しても手をつけない
→ 65歳以降は年金と組み合わせて長寿化
```

### 🔄 毎月の引き出しフロー

```
【月次の生活費調達手順】

Step 1：💴現金バケツ（普通預金）から当月生活費を引き出す
         └ 毎月定額（36.2万円）を引き出すだけ

Step 2：🔗繋ぎバケツの配当・分配金が自動入金（月1〜3回）
         └ 合計月約12.5万円が自動補充される
         └ 差額23.7万円が💴現金バケツから実質的に出る

Step 3：💴現金バケツ残高を確認
         └ 残高 ÷ 月支出 = 「残り何ヶ月分か」を計算
         └ 18ヶ月（1.5年）を切ったら🔗繋ぎバケツから補充トリガー発動
```

### 📅 年次補充・リバランス（毎年1月）

```
【年次作業フロー（所要時間：半日）】

① 📈 CPI確認と生活費の引き出し額をインフレ分だけ調整

② 💴 現金バケツの残高確認
   ├ 残高が2年分以上 → 何もしない
   └ 残高が2年分未満 → 🔗繋ぎバケツから補充

③ 🔗 繋ぎバケツの確認
   ├ 残高が目標の80%以上 → 🌱成長バケツから一部売却して補充
   └ 残高が目標の80%未満かつ相場が悪い → 💴現金バケツで耐える

④ 🌱 成長バケツの確認（相場状況で判断）
   ├ 相場が好調（直近高値から-10%以内）→ 必要額を売却
   └ 相場が暴落中（-20%以上）→ 売却禁止。💴・🔗で凌ぐ

⑤ 🔄 全体のリバランス
   └ 🌱成長バケツが総資産の60%超なら、一部を🔗繋ぎバケツへ
```
