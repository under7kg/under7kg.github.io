2026年7月、日経平均の年初来高値は6月22日の**72,831円**。三井住友DSアセットは5月の時点で「過去最大の上げ幅」「RSI70%超え」と警告を出していた。

これは2024年7月の高値42,426円から見れば、**2年で70%上昇**した計算になる。

この上昇を支えているのは何か。複数の要因が重なっている。新NISAの拡充による国内個人資金の流入、AI・半導体ブーム、高市政権の積極財政への期待、そして**円キャリートレードの資金**だ。

ただし、本稿で注目すべきはキャリートレードの資金だけだ。理由は他の要因と決定的に性質が違うからだ。NISA積立は現物の長期保有でパニック時にも動かない。AIブームは業績に裏打ちされてる。政治期待は政策次第で変動するが即座に市場から消えない。

**キャリートレードだけが「レバレッジ付きの短期資金」でパニック時に一斉に逃げる。**

大規模な巻き戻しは「日銀が止められない時」に来る。20〜40兆円規模の解消が現実のシナリオであり、日銀も政府も止められない。

ではその時、個人の資産はどうなるのか。本稿は資産別のインパクトを推定し、個人がどう備えるべきかの思考フレームを提示する。

**前提として述べておく。** タイミングを当てることは不可能だ。本稿の目的は「いつ来るか」ではなく「来た時にどう生き残るか」を考えることだ。

## 1. 大規模巻き戻しの前提条件

### 2024年8月との決定的な違い

| | 2024年8月 | 大規模巻き戻し局面 |
|---|---|---|
| 解消規模 | 4〜8兆円 | 20〜40兆円 |
| 日銀の対応 | 即座にハト派転換 | 転換不可（インフレ優先） |
| 回復期間 | 2週間 | 3〜6ヶ月以上 |
| 波及範囲 | 円・日本株・一部リスク資産 | 全リスク資産 |

### 三重のショック

大規模巻き戻し局面では以下の3つが**同時に**走る：

```
① キャリー解消 → 外国資金が一斉に引き上げ
② 円高急進 → 輸出企業の利益が圧迫
③ 金利上昇 → 割引率が上昇、バリュエーションが崩壊
```

2024年8月は①と②しか起きなかった。大規模巻き戻し局面では③が加わる。これが回復を遅らせる根本理由だ。

### 現在の過熱感

2026年7月時点で、日経平均は年初来高値72,831円を記録済み。三井住友DSアセットが5月に指摘した過熱感の指標は以下の通り：

| 指標 | 5月時点の状況 | 意味 |
|---|---|---|
| 1日上げ幅 | 3,320円（過去最大） | 2024年8月6日の3,217円を超過 |
| RSI | 70%水準 | 「買われすぎ」領域 |
| 25日線かい離率 | 5%超、10%に接近 | 過熱サイン |
| 押し上げ銘柄 | 半導体関連の値がさ株 | 特定セクターへの偏り |

つまり**大規模な巻き戻しが起きる前から、市場は過熱状態**にある。高値が高いほど、落差も大きくなる。


## 2. 資産別インパクト：短期・中期・長期

### 外国株（米株・新興国株）

| 期間 | 推定変動 | 理由 |
|---|---|---|
| 短期（1〜2週間） | -10〜20% | キャリー資金の強制解消 |
| 中期（1〜6ヶ月） | V字回復の可能性 | ファンダメンタルズは無傷 |
| 長期（6ヶ月〜） | 分化 | 米大型株は回復、新興国は遅れる |

**外国株は一時的な暴落から回復する可能性が高い。** 理由は米国のファンダメンタルズが変わらないからだ。キャリー資金が抜けても、米国内の資金は残る。

2024年8月の時も、S&P500は8月5日に-3%下落しただけで、2週間後に戻った。大規模巻き戻し局面では-10〜20%を想定すべきだが、回復の構造は同じと考えられる。

ただし**新興国株**は回復が遅れる可能性がある。円キャリーの主要な投資先だったため、資金流出の影響が長引く。

### 日本株

| 期間 | 推定変動 | 理由 |
|---|---|---|
| 短期（1〜2週間） | -25〜35% | 三重ショックの同時発生 |
| 中期（1〜6ヶ月） | 底に滞在 | 日銀が転換できない |
| 長期（6ヶ月〜） | 分化 | 内需株は復活、輸出株は構造転換 |

**日本株が最も深刻な影響を受ける可能性が高い。**

現在の69,744円（2026年7月）を前提に計算すると：

| シナリオ | 下落率 | 想定水準 |
|---|---|---|
| 軽度 | -25% | 約52,000円 |
| 中度 | -35% | 約45,000円 |
| 深刻 | -40% | 約42,000円 |
| 最悪 | -50% | 約35,000円 |

2024年8月は高値42,426円から31,156円まで-26%下落し、2週間で回復した。大規模巻き戻し局面では高値が72,000円台である上に、回復が3〜6ヶ月かかる。**落差も回復期間も桁違い**だ。

### 不動産

| 期間 | 推定変動 | 理由 |
|---|---|---|
| 短期（1〜3ヶ月） | 表面変動なし | 取引が停滞するだけ |
| 中期（3〜12ヶ月） | -10〜20% | 住宅ローン金利上昇で購買力低下 |
| 長期（1年〜） | 長期低迷 | 人口動態と金利の二重苦 |

**不動産は遅れて下落する可能性が高い。** 株式市場と違い、不動産は鈍重な資産だからだ。ただしJ-REITは即座に暴落する。

### 個人の資産別

| 資産 | 短期 | 中期 | 長期 |
|---|---|---|---|
| 現金・預金 | ✅ 安全 | ✅ 購買力向上 | △ 金利次第 |
| 外国株 | ❌ 暴落 | ✅ 回復 | ✅ 分化 |
| 日本株 | ❌ 壊滅 | ❌ 長期低迷 | △ 内需は復活 |
| 不動産 | △ 鈍重 | ❌ 下落開始 | ❌ 長期低迷 |
| 日本国債 | ❌ 価格下落 | ❌ 金利上昇 | △ 短期は有利 |
| 金・貴金属 | ✅ 上昇 | △ 調整 | ✅ 避難資産 |
| クリプト | ❌ 暴落 | △ 回復不明 | ❌ 不透明 |


## 3. 日本の資産が特に危険な理由

### 三重苦の構造

```
円高 → 輸出企業の利益圧迫
金利上昇 → 割引率上昇でバリュエーション崩壊
キャリー解消 → 外国資金が一斉に引き上げ
```

この3つが同時に走るのは**日本だけ**だ。米国は金利上昇の影響を受けるが、円高・キャリー解消の直接的影響は限定的だ。

### 外国資金の依存度

日経平均の上昇を支えてきたのは**外国投資家**だ。2023年の東証取引額に占める外国投資家の比率は約60%。この資金が一斉に引き上げられると、日本株は支える主体を失う。

### 現在の過熱感が意味するリスク

2026年7月時点で日経平均が72,000円台に乗せている状況は、複数の要因が重なった結果だ。新NISAによる国内個人資金の流入も確かに寄与してる。ただし、下落のトリガーはキャリーだけだ。

```
2024年7月：高値42,426円 → 8月に31,156円（-26%）
2026年7月：高値72,831円 → 大規模巻き戻し局面で？？
```

高値が70%上がったということは、キャリー資金の積み上がりも比例して大きくなってる可能性がある。解消規模も大きくなる可能性がある。

NISA積立が**下落を和らげるクッション**になる可能性はある。2024年8月のショックでも、個人投資家のNISA積立は止まらなかった。大規模巻き戻し局面でも同じ挙動が見込まれる。ただし**クッションであって防波堤ではない**。キャリー解消の規模が20〜40兆円なら、NISAの年間流入額（推定数兆円規模）では止めきれない。

### 不動産の特殊な危険性

日本の不動産は**住宅ローン金利に過敏**だ。

```
金利上昇 → 住宅ローン金利上昇
→ 購入者の購買力低下
→ 成約量減少
→ 価格下落開始
```

変動金利で住宅ローンを組んでいる個人は、金利上昇の直接的な打撃を受ける。固定金利を選択していても、新規購入者の購買力低下で自身の不動産価値が下がる。

### 長期低迷のリスク

日本の資産が長期低迷する理由は、**人口動態**だ。

- 購買力のある世代は減少中
- 不動産の需要構造が構造的に縮小
- 日本株の国内投資家基盤が弱い

大規模巻き戻しで下落した後、人口動態のテールウインドがないため、**回復が遅れる**可能性がある。


## 4. 資産サバイバルの3原則

### 原則①：日本の資産を減らす

```
日本株 → ウエイトを下げる
日本不動産 → 慎重な判断が必要
日本国債 → 短期のみ、長期は避ける
```

**理由：** 大規模巻き戻しの三重苦は日本の資産に集中する。リスクを分散するだけで、ダメージを大幅に軽減できる。

### 原則②：現金と海外資産を増やす

```
現金 → 円高で購買力が向上する
海外株 → 日本の三重苦の影響を回避
金・貴金属 → 避難資金が流入する
```

**理由：** 大規模巻き戻しで相対的に価値を保つ可能性が高いのは現金と避難資産だ。海外株は一時的な暴落から回復する可能性が高い。

### 原則③：レバレッジを排除する

```
住宅ローン → 固定金利に切り替え
FX証拠金 → ポジションを縮小
信用取引 → 解消する
```

**理由：** 大規模巻き戻し局面では追証の連鎖が起きる。レバレッジを持っている個人は、**システム的に強制決済される**。「待てば戻る」と判断しても、システムが勝手に売る。


## 5. 資産別の具体的対応策

### 現金・預金

**最も安全な資産の一つになる可能性が高い。**

- 円高で輸入物価が下落 → 実質購買力向上
- 金利上昇で定期預金の利回り向上
- 緊急時の流動性確保

**対応策：** 6ヶ月〜1年分の生活費を現金で保持。ただし**全財産を円現金にするのは危険**。インフレが進行する中で円の実質価値は下がるため。

**補助数値：** 現在の日本の家計の平均貯蓄額は約1,800万円（2人以上世帯・勤労者世帯）。ただし中央値は約1,000万円程度。生活防衛資金としては**300〜500万円**を最低限のラインとし、残りを投資に回す構造が理想だ。

### 外国株

**一時的な暴落は買い場となる可能性がある。**

- 短期的にはキャリー解消で暴落
- 中期的にはファンダメンタルズで回復
- 米大型株は特に回復が早い

**対応策：**
- 平時から一定のウエイトを維持
- 暴落時に追加購入する勇気を持つ
- 新興国株はウエイトを抑える

**補助数値：** S&P500の過去の暴落からの回復期間の参考データ：

| イベント | 最大下落率 | 回復期間 |
|---|---|---|
| リーマンショック（2008） | -57% | 約4年 |
| コロナショック（2020） | -34% | 約5ヶ月 |
| 2024年8月キャリー解消 | -3% | 約2週間 |

大規模巻き戻し局面では-10〜20%を想定すべきだが、ファンダメンタルズが無傷なら回復は数ヶ月以内に来る可能性が高い。

### 日本株

**最も慎重な判断が必要。**

- 短期的には壊滅的な暴落
- 中期的には長期低迷
- 長期的には内需株と輸出株で分化

**対応策：**
- 平時からウエイトを下げる
- 保有する場合は内需株中心に構成
- 輸出株は円高リスクを織り込む
- 暴落時に「底買い」するのはリスクが高い

**補助数値：** 現在の日経平均のPER（予想益倍率）は約17〜18倍。過去の底値圏のPERは10〜12倍。大規模巻き戻しでPERが12倍に収縮すると仮定すると、EPSが変わらない前提で**日経平均は30〜40%下落**する計算になる。現在の70,000円台なら42,000〜49,000円が底値圏の目安となる可能性がある。

### 不動産

**遅れて下落する可能性があるが、影響は長期に及びうる。**

- 短期的には取引停滞
- 中期的には価格下落開始
- 長期的には人口動態の逆風

**対応策：**
- 購入を急ぐ必要はない
- 住宅ローンは必ず固定金利
- 投資用不動産は慎重な判断が必要
- 都心の一等地以外は回避が無難

**補助数値：** 住宅ローン金利と購買力の関係：

| 住宅ローン金利 | 3,000万円借入の月返済額（35年） | 購買力指数 |
|---|---|---|
| 0.5%（現行水準） | 約7.7万円 | 100 |
| 2.0% | 約9.8万円 | 79 |
| 3.0% | 約11.4万円 | 68 |
| 4.0% | 約13.2万円 | 58 |

金利が4%になると、購買力は現在の58%に低下する。つまり**現在4,000万円のマンションを買える人が、金利4%では2,300万円のマンションしか買えなくなる**。これが不動産価格下落のメカニズムだ。

### 金・貴金属

**避難資金が流入し、上昇する可能性がある。**

- 短期的には混乱で買われる
- 円建て金価格は上昇
- 長期的には避難資産として機能

**対応策：**
- ポートフォリオの5〜10%を金で保持
- 物理的な金ではなく金ETFでも可
- 金利上昇環境では金利コストに注意

**補助数値：** 金価格は過去の危機時に以下の上昇を記録している：

| イベント | 金価格上昇率（3ヶ月） |
|---|---|
| リーマンショック（2008） | +20% |
| コロナショック（2020） | +15% |
| ロシア侵攻（2022） | +12% |

大規模な巻き戻しで円高とパニックの二重効果で、円建て金価格は+15〜25%の上昇が想定される。

### クリプト

**最も不透明な資産。**

- キャリー資金が流れていたため暴落
- 回復の不確実性が高い
- 規制リスクも加わる

**対応策：**
- 投機的なポジションは縮小
- ビットコイン以外はさらにリスクが高い
- 生活防衛資金をクリプトに入れない

**補助数値：** 2024年8月のキャリー解消時のクリプト下落：

| 資産 | 8月5日の下落率 |
|---|---|
| ビットコイン | -15% |
| イーサリアム | -20% |
| アルトコイン全体 | -25〜40% |

大規模な巻き戻しで2〜3倍の下落率を想定すべきだ。ビットコインで-30〜40%、アルトコインで-50〜70%が想定されるシナリオの一つだ。


## 6. タイミング対処の考え方

### タイミングを当てることは不可能

大規模な巻き戻しのタイミングを正確に予測できる人はいない。以下の指標を注視しつつ、**常に備えた状態を維持する**のが現実的だ。

### 注視すべき4つの指標

| 指標 | しきい値 | 意味 |
|---|---|---|
| 春闘賃上げ率 | 5%超が2年連続 | インフレ定着の可能性 |
| CPIコア | 3%超が半年継続 | 日銀の利上げ圧力が高まる |
| USD/JPY | 180超 | 円安の政治的限界が近い |
| 長期金利 | 2.5%超で定着 | 借り換え完了前の金利上昇 |

### 段階的な対応

```
通常期：日本株ウエイト30%以下、海外資産50%以上
警戒期：日本株ウエイト20%以下、現金比率30%以上
危険期：日本株をさらに縮小、現金と金を増やす
```

### 全有りか全無しかは最悪の選択

**中途半端な対応が一番危険**だ。大規模巻き戻しが来ると確信して全てを現金化するのも、来ないと確信して日本株全開にするのも、極端な賭けだ。

**段階的に調整する**のが最もリスクが低い。


## 7. 心理的準備：パニックに飲まれないために

### パニック時にやってはいけない3つのこと

**① 感情的に売る**
大規模巻き戻し局面の暴落時に感情的に売ると、**底値で売るリスクが高い**。2024年8月でも8月5日に売った人は大損した。

**② 「今回は違う」と思う**
「今回は違う」「日本は特別」は、バブル崩壊時の定型的な言葉だ。構造が同じなら結果も同じだ。

**③ レバレッジで取り返そうとする**
暴落後にレバレッジで「底買い」するのは、追証の連鎖に自ら飛び込む行為だ。

### パニック時にやるべき3つのこと

**① 事前に決めたルールに従う**
暴落が起きる前に「どこでどう動くか」を決めておく。パニック時に判断すると必ず間違える。

**② 現金の購買力を意識する**
円高で現金の実質購買力は上がる。資産価格が下がっても、相対的な豊かさは変わらないことがある。

**③ 長期的な視点を保つ**
大規模巻き戻しは数ヶ月〜1年の出来事だ。10年の視点で見れば、一時的な乱高下に過ぎない。ただし**日本の資産は長期低迷する可能性が高い**ため、長期保有の前提が崩れる点に注意。

円キャリートレードは、統計の死角に隠れた「見えない巨大資金」だ。その解消は、いずれ来る可能性がある。2024年8月は警告だった。大規模巻き戻しは、日銀が止められない時に来る。

現在の日経平均72,000円台は、複数の要因が作った高値だが、その中でキャリー資金だけが「逃げる資金」だ。インフレが定着し、日銀が利上げを止められなくなった時、この資金が一斉に引き揚げられる。

個人の対応策の方向性はシンプルだ。**日本の資産を減らし、現金と海外資産を増やし、レバレッジを排除する。** これだけでも、大規模巻き戻しのダメージを大幅に軽減できる。

問題は「来るかどうか」ではない。「来た時にどう生き残るか」だ。その準備を今から始めることが、唯一の資産サバイバルの道だ。
